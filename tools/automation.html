> Tools > Automation

This section includes automation tools and wrapper scripts for well-known and public security tools to make your life easier. You can adjust the scripts fast and easily according to your own needs. Mostly written in bourne shell.


Date File Description Version Author
03-25-2020 nullscan-1.0.0.tar.gz A modular framework designed to chain and automate security tests. It parses target definitions from the command line and runs corresponding modules and their nullscan-tools afterwards. It can also take hosts and start nmap first in order to perform a basic portscan and run the modules afterwards. Also, nullscan can parse a given nmap logfile for open tcp and udp ports and again run the modules afterwards. All results will be logged in specified directories with a clean structure and a HTML report can subsequently be generated.

1.0.0 noptrix
12-19-2017 wssh-0.1.tar.gz A shell script written with the purpose to automate jumps between boxes via ssh forgetting about IPs, Users, Ports and so on. You can set your host list at /home/${USER}/.wssh/wssh.conf file and start ssh'ing boxes, executing automated commands, or downloading/uploading files just using name/id associated.

0.1 nrz
02-22-2015 wnmap-0.1.tar.gz A shell script written with the purpose to automate and chain scans via nmap. You can run nmap with a custom mode written by user and create directories for every mode with the xml/nmap files inside. 0.1 nrz
